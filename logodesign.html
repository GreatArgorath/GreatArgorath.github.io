Libultraship Logo

Created using Illustrator
February 12th 2023

I created this cover because the official covers for these stories started becoming really formulaic, using the same background on each one, just swapping the characters around, so for my own collections and as something for the wider community I decided to create my own alternate covers.

This story had already been announced about a year prior, and I had created my own cover just based off of the title back then, but I knew I could do better now that we had more story details, so I started again, the only element that remained was the polaroids of the vampire teeth, although now swapped out for ones more fitting of the vampire in the actual story.
